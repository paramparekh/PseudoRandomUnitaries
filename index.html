Quantum Oracle Adversary ($\mathcal{A}$)

Security Definition: Let $\{\mathcal{U}_n\}_{n \in \mathbb{N}} = \{U_k\}_{k \in \mathcal{K}_n}$ be a family of $n$-qubit unitaries.

$$ \left|\, \underset{\mathcal{O} \leftarrow \mathcal{U}_n}{\mathbb{E}}\, \mathrm{Tr}\!\left(D_{\mathcal{A}} \cdot |\mathcal{A}^{\mathcal{O}}\rangle\!\langle \mathcal{A}^{\mathcal{O}}|_{\mathsf{AB}}\right) - \underset{\mathcal{O} \sim \mathrm{Haar}}{\mathbb{E}}\, \mathrm{Tr}\!\left(D_{\mathcal{A}} \cdot |\mathcal{A}^{\mathcal{O}}\rangle\!\langle \mathcal{A}^{\mathcal{O}}|_{\mathsf{AB}}\right) \right| \leq \mathrm{negl}(n) $$

Simulating Haar-Random Unitaries

How do we efficiently mimic a Haar-random $U$ for a querying quantum algorithm?

Solution: The Path-Recording Oracle

Use entanglement with a hidden data structure that "remembers" enough information as the adversary queries.

Up next: "Deriving" the path-recording oracle through simple examples

Example 1

Simulating the behavior of a single query $\lvert 0 \rangle$

Exact Simulation

Implementing a true Haar-random unitary $U$ is computationally infeasible.

$$U \sim \text{Haar}$$
$$\lvert 0 \rangle_a \longmapsto U \lvert 0 \rangle_a$$
(A maximally mixed state upon averaging)

Efficient Simulation

A QPT algorithm can efficiently realize the identical mixed state by entangling the query space $a$ with a hidden register $s$.

$$ \sum_{y} \lvert y \rangle_a \lvert y \rangle_s $$
Tracing out the hidden subsystem $s$:
The reduced state on $a$ is also a maximally mixed state.

Example 2

Simulating the behavior of two parallel queries $\lvert 0 \rangle_a$ and $\lvert 0 \rangle_b$

Exact Simulation

Evaluating parallel queries where $U \sim \text{Haar}$ is completely infeasible.

$$U \sim \text{Haar}$$
$$\lvert 0 \rangle_a \longmapsto U \lvert 0 \rangle_a, \quad \lvert 0 \rangle_b \longmapsto U \lvert 0 \rangle_b$$
(The joint state $U\lvert 0\rangle_a \otimes U\lvert 0\rangle_b$ is swap-symmetric maximally mixed)

Efficient Simulation

A QPT algorithm can efficiently realize the identical state by entangling the queries with a hidden structure that enforces swap symmetry.

$$ \sum_{y_1,y_2} \lvert y_1 \rangle_a \lvert y_2 \rangle_b \lvert \{y_1,y_2\} \rangle_s $$
Tracing out the hidden subsystem $s$:
The reduced state on $ab$ is also swap-symmetric maximally mixed.

Example 3

Simulating mixed queries $\lvert 0 \rangle_a$, $\lvert 0 \rangle_b$, and $\lvert 1 \rangle_c$

Exact Simulation

Evaluating mixed queries where $U \sim \text{Haar}$ is completely infeasible.

$$U \sim \text{Haar}$$
$$\lvert 0 \rangle_a \mapsto U \lvert 0 \rangle_a,\; \lvert 0 \rangle_b \mapsto U \lvert 0 \rangle_b,\; \lvert 1 \rangle_c \mapsto U \lvert 1 \rangle_c$$
($U\lvert 0\rangle_a$ & $U\lvert 0\rangle_b$ are swap-symmetric;
neither is symmetric with $U\lvert 1\rangle_c$)

Efficient Simulation

A QPT algorithm efficiently realizes this asymmetry by encoding ordered pairs in the hidden register to track which input label each query carried.

$$ \sum_{y_1,y_2,y_3} \lvert y_1 \rangle_a \lvert y_2 \rangle_b \lvert y_3 \rangle_c \lvert \{(0,y_1),(0,y_2),(1,y_3)\} \rangle_s $$
Tracing out the hidden subsystem $s$:
The reduced state is swap-symmetric between $a$ & $b$,
but neither is symmetric with $c$.

Formalizing the Path-Recording Oracle

We can achieve our simulation efficiently by replacing each query to $U$ with a lazily evaluated map $V$.

$$ V : \lvert x \rangle_A \lvert R \rangle_\mathsf{R} \mapsto \frac{1}{\sqrt{N - |R|}} \sum_{y \notin \mathrm{Im}(R)} \lvert y \rangle_A \, \lvert R \cup \{(x,y)\} \rangle_\mathsf{R} $$

The PRU Construction: $P_\pi \cdot F_f \cdot C$

A candidate PRU built from a permutation, a phase function, and a Clifford

Construction
Random Permutation $P_\pi$
$P_\pi = \displaystyle\sum_x |\pi(x)\rangle\langle x|$ ,   $\pi \leftarrow S_{2^n}$
Random Phase $F_f$
$F_f = \displaystyle\sum_x (-1)^{f(x)}|x\rangle\langle x|$ ,   $f \leftarrow \{0,1\}^{2^n}$
Random Clifford $C$
$C$ is a random $n$-qubit Clifford unitary

Generalization: After $t$ Queries

$$ |\Psi_t\rangle \propto \sum_{\vec{x}, \vec{y}} \Bigl( \prod_{i=1}^t |y_i\rangle\langle x_i| A_i \Bigr)_A |0^n\rangle \otimes \underbrace{ \sum_{\pi, f} \prod_{i=1}^t \delta_{\pi(x_i)=y_i} (-1)^{\sum f(x_i)} |\pi, f\rangle }_{\text{Consistent with } R = \{(x_i, y_i)\}} $$

For relation $R = \{(x_i,y_i)\}_{i=1}^t$:

$$ |\mathsf{pf}_R\rangle_{PF} \propto \sum_{\pi : R} |\pi\rangle_P \otimes \sum_{f} (-1)^{\sum_{(x,y)\in R} f(x)} |f\rangle_F $$

Final $t$-query state in compact form:

$$ |\Psi_t\rangle \propto \sum_{\vec{x},\vec{y}} \Bigl( \prod_{i=1}^t |y_i\rangle\langle x_i| A_i \Bigr)_A |0^n\rangle \otimes |\mathsf{pf}_{\{(\vec{x},\vec{y})\}}\rangle_{PF} $$

Intuition: Relation to Path-Recording Oracles

Key Observation: if all the $x_1, \cdots,x_t$ are distinct then $|\mathsf{pf}_{\{(x_1,y_1), \cdots, (x_t,y_t) \}}\rangle$ states are orthogonal and is isometric to $|\{(x_1,y_1), \cdots, (x_t,y_t)\}\rangle$.

Oracle $\mathsf{pf}O$

Acts on the purifying register $PF$:

$$ \mathsf{pf}O |x\rangle_A |\mathsf{pf}_R\rangle = \frac{1}{\sqrt{N-|R|}} \sum_{y} |y\rangle_A |\mathsf{pf}_{R \cup \{(x,y)\}}\rangle $$
Path-Recording Oracle $V$

Records classical set $R$:

$$ V |x\rangle_A |R\rangle = \frac{1}{\sqrt{N-|R|}} \sum_{y \notin \mathrm{Im}(R)} |y\rangle_A |R \cup \{(x,y)\}\rangle $$

Without access to the purifying registers, a query to random $P_\pi \cdot F_f$ is identical to $V$, except on paths with collisions among $x_1, \dots, x_t$.

$\implies$ We add $C$ to explicitly remove these collisions!

Core Equivalence
$$ \mathrm{Comp} \cdot \widetilde{\Pi}^{\mathrm{dist}}_{\mathrm{PF}} \cdot \bigl|\mathcal{A}^{\mathrm{pfO} \cdot U}_{t}\bigr\rangle_{\!\mathrm{ABPF}} = \Pi^{\mathrm{dist}}_{R^{(t)}_{X}} \cdot \bigl|\mathcal{A}^{V \cdot U}_{t}\bigr\rangle $$
Compression Operator
$$ \mathrm{Comp} := \sum_{R \,\in\, \mathcal{R}_{\mathrm{bij}}} |R\rangle\langle \mathrm{pf}_R| $$

Intuitively: "Compresses" the state $|\mathrm{pf}_R\rangle$ (requires exponential qubits) to $|R\rangle$ (size of the relation).

Path-Recording Projector

$\Pi^{\mathrm{dist}}_{R^{(t)}_X}$ projects onto the subspace in which all queried inputs are distinct.

Purified-Oracle Projector

$\widetilde{\Pi}^{\mathrm{dist}}_{\mathrm{PF}}$ projects onto the span of purified-oracle states whose recorded queries are all distinct.

Proof: Hybrid 2 $\approx$ Hybrid 3

By tracing out the oracle registers, we define the density matrices corresponding to our intermediate states:

$$ \begin{aligned} \rho_1^{(\mathfrak{D})} &:= \mathrm{Tr}_{\mathrm{PF}} \left( \mathop{\mathbb{E}}_{U \leftarrow \mathfrak{D}} |\mathcal{A}_t^{\mathrm{pfO} \cdot U}\rangle\langle\mathcal{A}_t^{\mathrm{pfO} \cdot U}|_{\mathrm{ABPF}} \right) \\[8pt] \rho_2^{(\mathfrak{D})} &:= \mathrm{Tr}_{\mathrm{PF}} \left( \widetilde{\Pi}^{\mathrm{dist}}_{\mathrm{PF}} \cdot \mathop{\mathbb{E}}_{U \leftarrow \mathfrak{D}} |\mathcal{A}_t^{\mathrm{pfO} \cdot U}\rangle\langle\mathcal{A}_t^{\mathrm{pfO} \cdot U}|_{\mathrm{ABPF}} \cdot \widetilde{\Pi}^{\mathrm{dist}}_{\mathrm{PF}} \right) \\[8pt] \rho_3^{(\mathfrak{D})} &:= \mathrm{Tr}_{R} \left( \Pi^{\mathrm{dist}}_{R^{(t)}_X} \cdot \mathop{\mathbb{E}}_{U \leftarrow \mathfrak{D}} |\mathcal{A}_t^{V \cdot U}\rangle\langle\mathcal{A}_t^{V \cdot U}|_{\mathrm{ABR}} \cdot \Pi^{\mathrm{dist}}_{R^{(t)}_X} \right) \\[8pt] \rho_4^{(\mathfrak{D})} &:= \mathrm{Tr}_{R} \left( \mathop{\mathbb{E}}_{U \leftarrow \mathfrak{D}} |\mathcal{A}_t^{V \cdot U}\rangle\langle\mathcal{A}_t^{V \cdot U}|_{\mathrm{ABR}} \right) \end{aligned} $$

$$ \begin{aligned} \rho_4^{(\mathfrak{D})} &= \mathop{\mathbb{E}}_{U \leftarrow \mathfrak{D}} \mathrm{Tr}_R\left(|\mathcal{A}_t^{V \cdot U}\rangle\langle\mathcal{A}_t^{V \cdot U}|_{\mathrm{ABR}}\right) \\ &= \mathop{\mathbb{E}}_{U \leftarrow \mathfrak{D}} \mathrm{Tr}_R\left(|\mathcal{A}_t^{V}\rangle\langle\mathcal{A}_t^{V}|_{\mathrm{ABR}}\right) \tag{5.10} \end{aligned} $$
Lemma 4.3 (Right unitary invariance)

For any $n$-qubit unitary $U_A$:

$$ |\mathcal{A}_t^{V \cdot U}\rangle_{\mathrm{ABR}} = \left(U_{R^{(t)}_{X,1}} \otimes \cdots \otimes U_{R^{(t)}_{X,t}}\right)|\mathcal{A}_t^{V}\rangle_{\mathrm{ABR}} $$
$$ \begin{aligned} \mathrm{Tr}_R\left(|\mathcal{A}_t^{V \cdot U}\rangle\langle\mathcal{A}_t^{V \cdot U}|_{\mathrm{ABR}}\right) &= \mathrm{Tr}_R\left(\left(U_{R^{(t)}_X}^{\otimes t}\right)|\mathcal{A}_t^{V}\rangle\langle\mathcal{A}_t^{V}|_{\mathrm{ABR}}\left(U_{R^{(t)}_X}^{\otimes t}\right)^\dagger\right) \\ &= \mathrm{Tr}_R\left(|\mathcal{A}_t^{V}\rangle\langle\mathcal{A}_t^{V}|_{\mathrm{ABR}}\right) \end{aligned} $$

Completion of the Proof

Combining our hybrid transitions and substituting $U = \text{Haar-random unitary}$ and $U = \mathfrak{D}$, we obtain the total trace distance between the construction and a truly random unitary oracle.

Efficient Circuit Implementation of the Path-Recording Oracle

Implement $V$ efficiently as a quantum circuit: $$ V : |x\rangle|R\rangle \;\mapsto\; \frac{1}{\sqrt{N-|R|}} \sum_{y \notin \mathrm{Im}(R)} |y\rangle \,|R \cup \{(x,y)\}\rangle $$
1 Store Relation $|R\rangle$

Sorted list of pairs in $|R| \cdot 2n$ qubits:

$|x_1\rangle|y_1\rangle \cdots |x_{|R|}\rangle|y_{|R|}\rangle$

with $(x_1,y_1) \leq \cdots \leq (x_{|R|}, y_{|R|})$ lexicographically

2 Superposition over Unused Outputs

Uniform superposition over indices, then apply bijection $f$:

$\frac{1}{\sqrt{N-|R|}} \sum_{k} |k\rangle \to$
$\frac{1}{\sqrt{N-|R|}} \sum_{y \notin \mathrm{Im}(R)} |y\rangle$

$f(k) = k + n_k \;,\; f^{-1}(y) = y - n_y$

$n_k$ = # used outputs $\leq k$
· computed via binary search on sorted $\mathrm{Im}(R)$

3 Update Record & Uncompute

Insert $(x, y)$ into sorted $R$, then erase scratch registers:

$|y\rangle|x\rangle|R\rangle|R\cup\{(x,y)\}\rangle$
$\mapsto |y\rangle\,|R\cup\{(x,y)\}\rangle$

Valid since $y \notin \mathrm{Im}(R) \Rightarrow (x,y)$ uniquely recoverable from $R\cup\{(x,y)\}$

  • ■ Efficient: O(log N) per step via binary search
  • ■ Reversible: All steps are invertible → valid unitary circuit
  • ■ Simulates Haar-random unitary up to inverse-exponential error ✓

Lemma 2.1 — Twirling and The Distinct Subspace

Setting: $n, t > 0$ integers, $\mathfrak{D}$ a unitary 2-design on $n$ qubits, $N = 2^n$, $\rho$ a state on $\geq nt$ qubits.

Distinct Subspace Projector: $$ \Pi^{\text{dist}} := \sum_{(x_1,\ldots,x_t)\in[N]^t_{\text{dist}}} |x_1\rangle\langle x_1|\otimes\cdots\otimes|x_t\rangle\langle x_t| $$

Claim: $$ \mathrm{Tr}\!\left(\underset{C\leftarrow\mathfrak{D}}{\mathbb{E}}\,(\Pi^{\text{dist}}\otimes\mathrm{Id})\cdot(C^{\otimes t}\otimes\mathrm{Id})\cdot\rho\cdot(C^{\dagger,\otimes t}\otimes\mathrm{Id})\cdot(\Pi^{\text{dist}}\otimes\mathrm{Id})\right) \;\geq\; 1 - \frac{t(t-1)}{N+1} $$

Key Ingredients

  • 1. Complement Projector: $\mathrm{Id} - \Pi^{\text{dist}} = \sum_{\text{non-distinct } \vec{x}} |\vec{x}\rangle\langle\vec{x}|$ (colliding tuples).
  • 2. Equality Projector: $\Pi^{\text{eq}} = \sum_{x\in[N]} |x,x\rangle\langle x,x|$ (projects onto pair collisions).
  • 3. Standard Twirling (Claim 1): $\underset{U\leftarrow\mathfrak{D}}{\mathbb{E}}\!\left[(U\otimes U)^\dagger\cdot\Pi^{\text{eq}}\cdot(U\otimes U)\right] = \frac{2}{N+1}\,\Pi^{N,2}_{\text{sym}}$.
  • 4. Quantum Union Bound: $\mathrm{Id} - \Pi^{\text{dist}} \;\preceq\; \sum_{i < j} \Pi^{\text{eq}}_{X_i, X_j}$ (failures bounded by sum of pair collisions).

Proof Steps

  1. Failure Probability Expansion: $$ 1 - \text{Success} = \mathrm{Tr}\!\left(\mathbb{E}_C (\mathrm{Id} - \Pi^{\text{dist}})\cdot C^{\otimes t}\cdot\rho\cdot C^{\dagger,\otimes t}\right) $$
  2. Applying Union Bound: $$ \leq \sum_{i < j} \underset{C\leftarrow\mathfrak{D}}{\mathbb{E}}\,\mathrm{Tr}\!\left(\Pi^{\text{eq}}_{X_i,X_j}\cdot C^{\otimes t}\cdot\rho\cdot C^{\dagger,\otimes t}\right) $$
  3. Tracing Out Non-Interacting Registers: $$ = \sum_{i < j} \underset{C\leftarrow\mathfrak{D}}{\mathbb{E}}\,\mathrm{Tr}\!\left(\Pi^{\text{eq}}_{X_i,X_j}\cdot (C \otimes C)\cdot\rho_{X_i,X_j}\cdot (C^\dagger \otimes C^\dagger)\right) $$
  4. Invoking Twirling Lemma: $$ = \sum_{i < j} \frac{2}{N+1} \mathrm{Tr}\!\left(\Pi^{N,2}_{\text{sym}} \cdot \rho_{X_i,X_j}\right) $$
  5. Final Bound: Since $\mathrm{Tr}(\Pi_{\text{sym}} \cdot \rho) \leq 1$ and there are $\binom{t}{2}$ pairs: $$ \text{Failure Prob} \leq \binom{t}{2} \cdot \frac{2}{N+1} = \frac{t(t-1)}{N+1} \qquad \square $$

Twirling scrambles the state such that collisions become as unlikely as in the classical Birthday Problem.
